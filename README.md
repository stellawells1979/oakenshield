

本项目组织了两个机器人，分别是搜索机器人，规则机器人，它们都有一个别名 search_bot,rules_bot
以供开发者要维护更新时易于阅读，机器人详功能细和参数，在 bot 文档中描述和定义

本项目依赖mysql数据库系统，所以首先要配置完整的mysql数据库系统

运行前需在 data 文件夹下创建 myaccount.txt 文件，并按以下格式输入内容
result_token = <规则机器人token>
search_token = <搜索机器人token>
host = <连接数据库的主机名>
port = <连接数据的端口>
user = <数据库用户名>
password = <数据库密码>
